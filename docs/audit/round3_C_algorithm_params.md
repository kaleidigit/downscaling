## C1 — Kaya γ_c 系数 [高] ✅ 已修复

### 问题

代码使用 **总量 GDP** 计算 γ，而 CLAUDE.md 和文献均要求 **人均 GDP**。

**修复前** (`common/downscale.py:148`):
```python
return 1.0 + 0.3 * np.log(gdp_c_2015 / gdp_world_2015)  # 总量
```

**修复后**:
```python
return 1.0 + 0.3 * np.log(gdp_pcap_2015 / gdp_pcap_world_2015)  # 人均
```

### 影响

| ISO | γ(修复前) | γ(修复后) | φ₂₀₅₀(修复前) | φ₂₀₅₀(修复后) |
|-----|----------|----------|-------------|-------------|
| chn | 0.51 | 1.01 | 0.28 | 0.48 |
| usa | 0.46 | 1.39 | 0.25 | 0.59 |
| nld | **-0.41** | 1.40 | **-0.30** | 0.59 |
| nga | **-0.53** | 0.58 | **-0.40** | 0.31 |
| deu | **-0.03** | 1.31 | **-0.02** | 0.57 |

- 修复前: **168/172 国 γ < 0**（收敛方向反转）
- 修复后: **1/172 国 γ < 0**（仅一个极端低人均 GDP 国家）

### 文献

- 0.3 系数: **未在 van Vuuren 2007 或 Gidden 2019 中找到确切出处**
- 标注为"校准参数"，建议在论文中进行 0.1/0.3/0.5 敏感性分析

### 敏感性（0.3 → 0.1/0.2/0.5）

对主要国家 φ₂₀₅₀ 的影响:
```
Multiplier:  0.1     0.2     0.3     0.5
China:      0.166   0.321   0.475   0.725
USA:        0.212   0.400   0.588   0.810
India:      0.116   0.248   0.380   0.603
```

0.5 系数使富国几乎完全收敛（φ≈0.8），0.1 系数使收敛非常缓慢（φ≈0.16）。

## C3 — Logit 份额迭代参数 [中] ✅

### 参数

| 参数 | 值 | 位置 |
|------|---|------|
| ε (epsilon) | 1e-4 | `downscale.py:536` |
| tol | 1e-6 | `downscale.py:538` |
| max_iter | 50 | `downscale.py:537` |

### 评估

- ε=1e-4: 合理。避免 log(0) 和 log(1/0) 数值问题
- tol=1e-6: 足够严格。迭代缩放误差 < 0.0001% 时退出
- max_iter=50: 充足。大多数情况在 5-10 次迭代内收敛

### 验证

Logit 份额计算的标准收敛行为：
1. 初始比例分配各国家
2. 缩放因子 k = target/sum 调整
3. 封顶溢出的国家，重新分配剩余量
4. 2-3 次迭代即可满足 tol

参数选择合理 ✅

## C5 — EI 趋势截断 [低] ⚠️

### 参数

```python
# io.py:591
cagr = max(-0.05, min(0.02, cagr))  # [-5%, +2%] 年变化率
```

### 实际分布

基于 2015-2020 IEA 数据计算的 149 国 EI 趋势:

| 统计量 | 值 |
|--------|---|
| 总国家数 | 149 |
| 截断于 -5%（下界）| **36** (24%) |
| 截断于 +2%（上界）| **1** (<1%) |
| 零趋势（无数据）| 7 (5%) |
| 中间范围（未截断）| 105 (70%) |

非零趋势分布: `Min=-0.050, Q25=-0.050, Median=-0.033, Q75=-0.014, Max=0.020`

### 问题

- **36 个国家被截断于 -5% 下界**，包括 **中国（chn）、印度（ind）** 等主要经济体
- Q25 恰好为 -0.050，说明下界设定与数据分布边缘重叠
- -5% 下界可能对快速工业化国家过于严格（这些国家的能源强度确实在快速下降）

### 建议

1. 保持 [-5%, +2%] 作为默认值，但标记为可配置参数
2. 在论文中报告被截断国家的数量和名称
3. 可选：对 Top 10 经济体使用原始趋势（不做截断）
