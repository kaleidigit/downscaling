# Plot 说明文档

18 张图表，分为 5 类。

---

## 1. 仪表板 — `dashboard_{indicator}_all.png`（8 张）

**每张图**：单指标 4 情景 2×2 布局。

| 子图位置 | 内容 |
|---------|------|
| 左上 | SSP1-2.6 (Sustainability) |
| 右上 | SSP2-4.5 (Middle Road) |
| 左下 | SSP4-3.4 (Inequality) |
| 右下 | SSP4-6.0 (Regional Rivalry) |

**每条曲线**：全球国家合计值的时序变化（2015-2100）。

**三种颜色**：
- 蓝色 = Logit（恒定份额比例法）——基准参照
- 橙色 = Kaya（van Vuuren 收敛法）——快收敛
- 绿色 = DSCALE（双路径法）——慢收敛

**右上角灰色直方图**（右轴）：Kaya 相对于 Logit 在 2100 年的国家层面百分比偏差分布。峰值偏离 0 越远，两方法差异越大。

**解读要点**：同色曲线在 4 个子图中高度相似但不等 → 情景影响显著。三色曲线分离度 → 方法差异大小。

## 2. 份额对比 — `share_{indicator}_all.png`（4 张）

**每张图**：单份额指标 4 情景 2×2 布局。

| 子图元素 | 内容 |
|---------|------|
| 曲线 | Top-5 国家份额时序，每国三条线（三方法） |
| Y 轴 | 0–1（份额有界指标）或 log 尺度（无界指标） |
| 左上角标注 | 可用方法列表（通常 Logit + Kaya + DSCALE） |

**解读要点**：三条线分离 → 方法对份额分配有不同判断。曲线趋势方向 → 份额上升或下降。注意：`electrification_rate` 和 `green_elec_share` 的 Kaya/DSCALE 值可能因收敛动力学与 Logit 产生较大偏离。

## 3. 收敛特性 — `convergence_profile.png`（1 张）

**三面板诊断图，揭示方法内在数学行为。**

| 面板 | 内容 | 解读 |
|------|------|------|
| A. Kaya 收敛速度 | 按 GDP 分位的 w(t) 曲线 | 曲线越陡→收敛越快。所有国家共用同一 γ（情景级），到 2100 年仅完成 ～30% 收敛（SSP126, y_c=2150） |
| B. DSCALE 转换 | 三种参数组合的 ENSHORT→ENLONG 转换 | 早期（低 MAX_TC）→ 快切换到区域 ENLONG；晚期（高 MAX_TC）→ 长期保留国家 ENSHORT |
| C. 方法分歧度 | 全球 TFC 总量层面 Kaya/DSCALE 相对 Logit 的偏差 | 偏差随时间扩大→方法差异累积 |

**重要**：面板 A 显示 Kaya 在 2015-2100 期间远未完成收敛（w(2100) ≈ 0.3 for SSP126）。这意味着即使在 2100 年，国家基年特征仍然主导 Kaya 输出。这是方法设计特性，不是 bug。

## 4. 散点图 — `compare_scatter_{scenario}.png`（4 张）

**每张图**：TFC 三方法两两散点，2 行 × 3 列。

| 行 | 年份 | 列 | 对比 |
|----|------|----|------|
| 上 | 2050 | Logit-Kaya, Logit-DSCALE, Kaya-DSCALE |
| 下 | 2100 | 同上 |

- 每个点 = 一个国家
- 对角线虚线 = 两方法完全一致
- 右上角 r 值 = log-log Pearson 相关系数（r 越接近 1 → 方法间排序越一致）

**解读要点**：上→下行对比看时间演变——2100 通常比 2050 分散（方法差异随时间累积）。Logit-DSCALE 通常最分散（两端方法差异最大）。偏离对角线的离群点 → 该国的方法敏感度高。

## 5. 全局汇总 — `global_summary.png`（1 张）

**五面板总图。一个文件了解项目全局。**

| 面板 | 内容 |
|------|------|
| A. 指标变化 | 8 个指标 × 3 方法，2100/2015 比值（柱状图），>1=增长，<1=下降 |
| B. 相关性热力图 | 8×8 指标间 log-log 相关系数矩阵（Logit 方法，2050 年），红色=正相关，蓝色=无关 |
| C. TFC 情景对比 | 全球 TFC 总量 2100 年跨 4 情景折线，每种方法一条线 |
| D. Top-10 份额 | Logit 方法 SSP126 下 2100 年全球 TFC 份额排名（水平柱状图） |
| E. 收敛权重 | 四种情景的 Kaya w(t) 曲线对比，虚线=99%收敛线（y_c 处） |

**解读要点**：
- A：CO2 排放在 SSP126 下降最显著（负排放情景），TFC 在 SSP245 最高
- B：TFC/电力/TES 高度相关（~0.9+），CO2 与其他指标相关性较低
- C：SSP245（中等发展路径）全球 TFC 最高，不同方法对情景排序一致
- E：收敛年份越远（SSP460→2300），2100 年完成比例越低
